Global Property Search Platform

Overview

Chinese property searching platform Juwai.com helped many Chinese buyers to find their dream home around the world .

In 2019, we expanded our business from China to Asia, by creating another real estate search platform, Juwai.asia, for Asian buyers.

Role: 50% UX Research，100% UI Design

Requirements & Constraints

Below are the requirements of the global Property Search Platform - Juwai.asia:

  • Implement main features like Juwai.com: property searching, global real estate market news, consulting services

  • The content of homepage must be customised according to user location

  • The UI needs to support multiple languages and currencies.

  • Users can search for local property

  • Responsive website

  • Needs to launch in 3 months

Because Juwai.com is not a responsive website, we had to develop everything from raw. Meanwhile, our target audience expanded from Chinese to Asian, we might need more than 3 months to get the first launch ready. Fortunately, we got approval to do work around.

In order to successfully release the initial version in 3 months, we worked on the most important features and workflow first.

Information Architecture

The Information Architecture is more or less similar to Juwai.com. It mainly serve the purpose for stakeholders and our team to keep track on the status of each page to make sure the project is on schedule. 

Design System

The UI of Juwai Asia is basically unified with Juwai.com in order to retain the brand positioning. The framework of the website is also based on Bootstrap.

Global Gateway

The content, currency and language of Juwai.asia must be automatically converted according to user's location, so we need to confirm where they are when they land to our site.
