Chinese Real Estate Mobile App Revamp

Juwai.com is an online property searching platform for Chinese real estate buyers to explore oversea properties. The project aims to improve the overall performance of the existing app.

Duration of Project

1 month

Tools

Sketch, InVision, Adobe AI

Role

UX & UI Designer

Platform

Mobile App

Study the problem of the existing app

We interviewed few users and property agents and observe how they use the app in the beginning, here are the main problems:

• Slow speed
• Insufficient features to help decision making
• Insufficient market info compared to PC site
• Outdated user interfaces

Persona and User Journey

4 personas was created based on the user research.
They are user groups with different motivation of searching oversea property listings.
