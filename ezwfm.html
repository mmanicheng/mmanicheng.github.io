Workforce Management System

ezWFM means Easy Workforce Management which is a plugin of Kronos Workforce Central. It allows employee to check working schedule and punch records on the mobile app, also submit leave request. At the same time, manager can approve leave via the app or web platform.

Duration of Project

On-going Project

Tools

Sketch, Balsamiq Wireframes

Role

UX & UI Designer

Platform

Mobile App, Web

Project Background

Before this project, an airline cargo company purchased a workforce management system which can set up complicated logic to manage their part-time staff's roster. However, after they use the new system for awhile, they have some other needs that the system cannot solve:

  • No mobile app
    HR has to print the roster after every edits but the part time staff always swap their shift with other coworkers. This ongoing edits burden HR's workload. 
  • Cannot setup complex leave logics
    The company has complicated rules and conditions to different leave types but the system can only setup simple logics.
  • Steep learning curve
    The system is slow and the interface is complicated for HR and managers who have been using Excel to manage workforce for years.

Discovery Stage

We arranged focus group meeting every week at the cargo office to learn about their staff aka. the users and their needs. We used diagrams to consolidate and analyse the data collected.
